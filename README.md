# Supermarket Sales Dashboard

This project is an interactive Power BI dashboard designed to analyze supermarket sales data and provide valuable business insights. The dashboard helps track key performance indicators such as total sales, profit, customers, and orders.

## Features
- Sales, profit, customer, and order KPIs
- Category-wise sales and profit analysis
- Monthly profit trends
- Quantity analysis by product categories
- Payment mode analysis
- Interactive filters for City, State, and Sub-category

## Technologies Used
- Power BI
- Data Visualization
- Business Intelligence

## Project Objective
The main objective of this project is to help businesses understand sales performance, identify profitable categories, analyze customer behavior, and make data-driven decisions through interactive visualizations.
